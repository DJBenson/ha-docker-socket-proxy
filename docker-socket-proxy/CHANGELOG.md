# Changelog

## 1.1.1

- Added translations for German, French, Spanish, Italian, Dutch, and Portuguese

## 1.1.0

- All Docker API options now configurable via addon settings
- Added UI labels and descriptions for all options
- Safe options prefixed with "Safe:", dangerous options prefixed with "DANGER:"
- Updated documentation to reflect configurable options

## 1.0.0

- Initial release
- Based on Tecnativa/docker-socket-proxy
- Configurable port option
- Read-only access to containers, images, networks, volumes
- Secure defaults blocking all write operations
